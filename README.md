# 🛒 Ishop - Full Stack Ecommerce Project

A full stack ecommerce web application built with Node.js, Express, PostgreSQL and Bootstrap.

## 🌟 Features

- ✅ User Registration & Login
- ✅ Customer Dashboard
- ✅ Product Listing (FakeStore API)
- ✅ Product Details Page
- ✅ Product Search
- ✅ Category Filtering
- ✅ Shopping Cart
- ✅ Quantity Update
- ✅ Wishlist
- ✅ Checkout Form
- ✅ Order Summary
- ✅ Payment UI
- ✅ Admin Panel

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, Bootstrap 5, jQuery, AJAX
- **Backend:** Node.js, Express.js
- **Database:** PostgreSQL
- **API:** FakeStore API
